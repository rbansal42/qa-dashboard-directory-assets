2.2 Measures · paste each into DashboardDirectory

Right-click DashboardDirectory in the Data pane → New measure → paste each block below.

TotalDashboards = COUNTROWS(DashboardDirectory)

NewThisMonth =
CALCULATE(
    COUNTROWS(DashboardDirectory),
    DashboardDirectory[CreatedOn] >= TODAY() - 30
)

FreshPct =
DIVIDE(
    COUNTROWS(
        FILTER(
            DashboardDirectory,
            SWITCH(
                DashboardDirectory[RefreshFrequency],
                "Real-time", (NOW() - DashboardDirectory[LastRefresh]) * 24 * 60 <= 5,
                "Hourly",    (NOW() - DashboardDirectory[LastRefresh]) * 24 <= 2,
                "Daily",     (NOW() - DashboardDirectory[LastRefresh]) <= 1.1,
                "Weekly",    (NOW() - DashboardDirectory[LastRefresh]) <= 8,
                TRUE
            )
        )
    ),
    COUNTROWS(DashboardDirectory)
)

PastSLACount =
COUNTROWS(DashboardDirectory) - [FreshPct] * COUNTROWS(DashboardDirectory)

ResultsCount =
VAR _cat = SELECTEDVALUE(DashboardDirectory[Category], "All areas")
VAR _n   = COUNTROWS(DashboardDirectory)
RETURN _n & " dashboards in " & _cat

LastRefreshFmt = "Last refresh: " & FORMAT(NOW(), "dd mmm yyyy, HH:MM")

Test & publish 13

13.1 Smoke test

  1. Click every category pill in the nav strip. The Matrix, Word Cloud, KPI cards and Most Popular should all filter visibly.
  2. Type otp into the banner search. Matrix narrows to OTP-related rows; count card updates.
  3. Click the OTP example chip. Search box populates with "otp". Matrix narrows.
  4. Click a Dashboard name in the matrix. Browser opens the URL.
  5. Click a pinned tile. Browser opens its URL.
  6. Click the "All Areas" pill. Everything resets.
  7. Change the Sort-by to "Name A–Z". Matrix re-sorts.
If a visual doesn't filter You probably have an Edit interactions setting wrong. Open it, select the misbehaving source visual, and check the target's icon — three small icons appear on every other visual: Filter / Highlight / None.

13.2 Publish

  1. Home → Publish → choose destination workspace
  2. In the Power BI service, pin the page to a dashboard if you want a shortcut tile
  3. Share the report link

13.3 Save as template

File → Export → Power BI template (.pbit). Future directory pages can open this template, swap in new data, and inherit every visual.

Appendix C · Padding & margins

Power BI doesn't expose CSS-style margins. Every gap between visuals is created by the X/Y coordinate you set. Every "padding" is either a text-padding property on a Shape/Button/Matrix, or extra whitespace around content inside a Text box. Here's a consolidated reference for both.

Reading the page top-to-bottom, these are the vertical gaps. Each is produced by the next element's Y coordinate being exactly X pixels below the previous element's bottom edge.

C.7 Quick rules of thumb

When positioning a new element
  • Panel edge → content: 24–28 px always — don't let anything touch the panel border.
  • Row-to-row vertical gap: 20 px is the default. Bump to 24 when moving between logically different zones (e.g. above the gallery strip).
  • Card-to-card horizontal gap: 10 px for dense rows (KPI, gallery), 18 px for the main column grid.
  • Inside a card: 14–16 px on all four sides. Less makes it feel cramped, more wastes space.
  • Pill padding: 4 × 12 px minimum (5 × 14 for larger pills). Anything smaller clips descenders.
  • Section heading → first row of content: 10 px, always with a 1 px divider on the heading's bottom border.

SymptomFix
Dashboard column shows plain text, not a link.Re-apply Column tools → Data category → Web URL to the URL column. Then re-apply conditional formatting → Web URL on the Dashboard column.
Clicking a category pill does nothing.You built it as individual buttons, not a Tile slicer. Replace with a single Slicer visual on Category with Tile style, single-select, show "Select all".
Example chip pushes nothing into search.Button action → Type must be Set slicer, not Page navigation. Destination must be the search slicer. Value must be the literal string.

Fresh % shows blank.The measure expects LastRefresh as datetime. Check the column's data type in Power Query; change to Date/time if it's text.
