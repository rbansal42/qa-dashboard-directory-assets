# Qatar Airways · Find Your Dashboard — Power BI design assets

Design assets for reproducing the "Find Your Dashboard" directory page in Power BI.

Companion tutorial: see the accompanying Word doc.

## Files

| File | Use |
|---|---|
| `qatar-theme.json` | Report theme — import via **View → Themes → Browse** |
| `qatar-wallpaper-1280x820.png` | Canvas background — **Format page → Canvas background → Image** |
| `dax-measures.txt` | DAX measures for the results count and last-refresh label |
| `sample-data.csv` | 13-row sample of the `DashboardDirectory` table |
| `qa-compass-portal-mockup.html` | Static HTML mockup of the target design |
| `wallpaper-source.html` | Source of the wallpaper PNG (in case you want to tweak) |

Canvas size: **1280 × 820 px** (custom). Fonts: Segoe UI (body) + Georgia / Cormorant Garamond (serif titles).
